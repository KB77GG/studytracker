# 小程序重新提审说明

## 建议填写的审核备注

本版本已提供无需登录、无需手机号、头像或昵称授权的完整演示入口。审核路径：打开小程序 -> 点击首屏主按钮“免登录体验学生端”；也可在首屏直接选择“家长端”或“教师端”，浏览任务、学习数据、家长报告、课堂反馈和教师课表的只读示例。游客浏览不会读取或保存个人数据。仅当用户主动使用任务提交、计时、账号绑定或消息提醒等需要保存个人数据的功能时，才会提示用户自行选择登录，提示框可取消并继续浏览。

## 提审前录屏路径

1. 清除开发者工具缓存，从冷启动打开小程序。
2. 展示首屏“免登录体验学生端”位于第一屏，且未出现授权弹窗。
3. 进入学生端，打开一条任务详情，再进入“我的”查看演示学习数据。
4. 返回首屏，分别进入家长端查看报告/反馈、教师端查看课表。
5. 点击一个需保存数据的操作，展示可取消的登录提示，并点击“继续浏览”。

将这段完整录屏及学生端、家长端、教师端截图上传到提审页面。该审核路径不需要测试账号或特殊环境配置。
